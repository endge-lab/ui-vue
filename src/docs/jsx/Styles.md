# Общие атрибуты отступов и рамок (Spacing / Border)

Любой компонент DSL в системе **Endge** поддерживает универсальные атрибуты для задания **отступов**, **рамок** и **радиусов**.  
Это позволяет гибко управлять внешним видом элементов без необходимости писать дополнительные CSS-правила.

---

## Отступы (Margin / Padding)

### Поддерживаемые атрибуты

| Атрибут | Описание                       | Пример               |
|---------|--------------------------------|----------------------|
| `p`     | Внутренние отступы (padding)   | `p="1"`, `p="0.5rem"` |
| `px`    | Горизонтальные паддинги        | `px="1"`             |
| `py`    | Вертикальные паддинги          | `py="0.5"`           |
| `pt`    | Верхний паддинг                | `pt="1"`             |
| `pb`    | Нижний паддинг                 | `pb="0.5"`           |
| `pl`    | Левый паддинг                  | `pl="1"`             |
| `pr`    | Правый паддинг                 | `pr="1"`             |
| `m`     | Внешние отступы (margin)       | `m="2"`              |
| `mx`    | Горизонтальные маргины         | `mx="1"`             |
| `my`    | Вертикальные маргины           | `my="0.5"`           |
| `mt`    | Верхний маргин                 | `mt="1"`             |
| `mb`    | Нижний маргин                  | `mb="0.5"`           |
| `ml`    | Левый маргин                   | `ml="1"`             |
| `mr`    | Правый маргин                  | `mr="1"`             |

⚙️ **Числовые значения** автоматически интерпретируются как `rem`.
`p="1"` = `padding: 1rem`  
⚙️ Можно использовать **любой CSS-формат** (`px`, `%`, `em`, `vh` и т.д.)

---

## Рамки (Border)

### Базовые шорткаты

| Атрибут | Область | Пример значения | Результат |
|----------|----------|----------------|------------|
| `b`      | Все стороны | `"1"`, `"1 #ddd"`, `"2 dashed #999"` | `border: 1px solid #ddd` |
| `bx`     | Лево и право | `"1 #eee"` | `border-left/right` |
| `by`     | Верх и низ | `"1 dashed #ccc"` | `border-top/bottom` |
| `bt` / `bb` / `bl` / `br` | Конкретная сторона | `"2 red"` | `border-top: 2px solid red` |

🔹 Форматы значений:
- `b="1"` - `1px solid currentColor`
- `b="2 #aaa"` - `2px solid #aaa`
- `b="1 dashed #ddd"` - `1px dashed #ddd`

---

### Атомарные настройки

| Атрибут | Что делает | Пример | CSS |
|----------|-------------|--------|------|
| `bw`     | Толщина рамки (все стороны) | `bw="2"` | `border-width: 2px` |
| `bs`     | Стиль рамки | `bs="dashed"` | `border-style: dashed` |
| `bc`     | Цвет рамки | `bc="#ddd"` | `border-color: #ddd` |
| `bwT` / `bwR` / `bwB` / `bwL` | Толщина по стороне | `bwT="4"` | `border-top-width: 4px` |
| `bsT` / `bsR` / `bsB` / `bsL` | Стиль по стороне | `bsR="dotted"` | `border-right-style: dotted` |
| `bcT` / `bcR` / `bcB` / `bcL` | Цвет по стороне | `bcB="red"` | `border-bottom-color: red` |

---

### Радиусы (скругления)

| Атрибут | Область | Пример | CSS |
|----------|----------|--------|------|
| `r`      | Все углы | `r="8"` | `border-radius: 8px` |
| `rTL` / `rTR` / `rBR` / `rBL` | Отдельные углы | `rTR="12px"` | `border-top-right-radius: 12px` |

🔹 Числовые значения интерпретируются как `px`.  
🔹 Можно использовать любые единицы (`rem`, `%`, `em`, `px`).

---

## Динамические значения

Все атрибуты можно использовать с `v-bind` и объектными выражениями:

```jsx
<Box :b="row.border" :r="radius" :="{ bw: 2, bc: theme.border, bs: 'dotted' }" />
<Row :="{ bt: '1 #eee', bcL: 'red', bwL: 3 }" />
<Card bx="2 dashed #999" :bb="0" :rTL="'12px'" />
