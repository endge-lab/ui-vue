## Text

Компонент `<Text>` используется для стилизации и отображения текста в DSL-описаниях.
Он поддерживает **статические** и **динамические (`v-bind`)** атрибуты, управляющие визуальными свойствами текста:
жирность, курсив, подчёркивание, цвет, фон, размер, регистр и выравнивание.

---

## Поддерживаемые атрибуты

| Атрибут        | Описание                                                   | Пример значения                |
|----------------|------------------------------------------------------------|--------------------------------|
| **`bold`**        | Делает текст жирным                                        | `true` (или просто `bold`)      |
| **`italic`**      | Делает текст курсивным                                    | `true`                         |
| **`underline`**   | Подчеркивает текст                                        | `true`                         |
| **`strike`**      | Зачёркивает текст                                         | `true`                         |
| **`color`**       | Цвет текста                                               | `#d32f2f`, `red`, `var(--text)`|
| **`bg`**          | Цвет фона текста                                          | `#f0f0f0`, `rgba(0,0,0,0.1)`   |
| **`size`**        | Размер шрифта. Число автоматически конвертируется в `px`  | `14`, `1rem`                   |
| **`align`**       | Выравнивание текста                                       | `left`, `center`, `right`      |
| **`weight`**      | Толщина шрифта                                            | `400`, `600`, `bold`           |
| **`uppercase`**   | Преобразует текст в ВЕРХНИЙ РЕГИСТР                       | `true`                         |
| **`lowercase`**   | Преобразует текст в нижний регистр                        | `true`                         |
| **`capitalize`**  | Каждое слово начинается с заглавной буквы                 | `true`                         |

> Атрибуты можно комбинировать произвольно - они не конфликтуют между собой.
> Булевые атрибуты можно указывать без значения: `<Text bold italic>...</Text>`.

---

## Динамические значения

Каждый атрибут может быть вычисляемым, если указать его через двоеточие (`:`):

| Пример | Описание |
|--------|-----------|
| `<Text :bold="isActive" />` | Применяет жирность, если `isActive === true` |
| `<Text :color="theme.primary" />` | Устанавливает цвет из реактивного контекста |
| `<Text :size="fontSize" />` | Преобразует числовое значение `fontSize` в пиксели |
| `<Text :="{ bold: isActive, color: theme.text, size: 14 }" />` | Объектное связывание нескольких свойств сразу |

Внутри выражений доступны переменные из текущего `scope` и `comData`.

---

## Пример использования

```jsx
<Text bold color="#d32f2f" size="16" underline>
  Привет, {{ user.name }}
</Text>

<Text :bold="user.isVip" :color="user.color" :size="fontSize" italic>
  {{ user.nickname }}
</Text>

<Text :="{ bold: active, color: theme.text, uppercase: caps }">
  {{ message }}
</Text>
```
