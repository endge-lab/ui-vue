# Атрибуты `if` и `else`

Любой компонент DSL поддерживает условный рендеринг с помощью атрибутов **`if`** и **`else`**. Это позволяет динамически скрывать или показывать элементы в зависимости от заданных условий.

## Поддерживаемые атрибуты

| Атрибут | Описание                                             | Пример                         |
|---------|------------------------------------------------------|--------------------------------|
| `if`    | Выражение, которое должно быть `true`, чтобы элемент отображался | `if="{ user.isActive }"`       |
| `else`  | Специальный маркер, который указывает альтернативный блок при `if=false` | `else` |

## Пример использования

```jsx
<Text if="{ user.isActive }" color="green">Пользователь активен</Text>
<Text else color="red">Пользователь не активен</Text>
