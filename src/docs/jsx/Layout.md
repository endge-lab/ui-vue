# Layout

Компонент **Layout** позволяет создать гибкую сеточную структуру для отображения дочерних элементов. Он поддерживает различные варианты выравнивания, отступы, а также адаптивное поведение.

## Основные свойства

| Свойство   | Тип     | Описание                                             | По умолчанию |
|------------|---------|------------------------------------------------------|--------------|
| `direction`| string  | Направление расположения элементов (`row`, `column`) | `row`        |
| `gap`      | string  | Отступ между элементами (например, `1rem`)           | `0`          |
| `align`    | string  | Выравнивание элементов вдоль оси `cross` (`center`, `start`, `end`) | `start` |
| `justify`  | string  | Выравнивание элементов вдоль главной оси (`center`, `space-between`, и др.) | `start` |

## Пример использования

```jsx
<Layout direction="row" gap="1rem" align="center" justify="space-between">
  <Button label="Назад" />
  <Button label="Далее" />
</Layout>
