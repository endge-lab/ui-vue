# Box

Компонент `<Box>` - это **пустой контейнер** для обёртки других элементов.

---

## Особенности

Работает как обычный контейнер (`div`)
Позволяет группировать другие элементы DSL
Поддерживает любые **общие стилистические атрибуты** (например, `p`, `m`, `bg`)

---

## Пример использования

```jsx
<Box>
  <Text>Это текст внутри Box</Text>
</Box>

<Box p="2" bg="#f0f0f0">
  <Text>Box с отступами и фоном</Text>
</Box>
