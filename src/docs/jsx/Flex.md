# Flex

Компонент **Flex** - это упрощённая версия базового Layout, которая помогает быстро создавать вертикальные (`col`) или горизонтальные (`row`) контейнеры с базовыми свойствами выравнивания и отступа.

По умолчанию:
- **Направление** - горизонтальное (`row`)
- **Выравнивание** второго направления - всегда `center` (в зависимости от оси)
- **Отступ между элементами** - `1rem`

## Атрибуты

| Атрибут  | Тип       | Описание                                                                 | Пример                  |
|----------|-----------|--------------------------------------------------------------------------|--------------------------|
| `col`    | `boolean` | Устанавливает вертикальное направление                                   | `<Flex col>`            |
| `row`    | `boolean` | Устанавливает горизонтальное направление (по умолчанию)                   | `<Flex row>`            |
| `align`  | `string`  | Выравнивание элементов по основной оси                                   | `"start"`, `"center"`, `"end"`, `"between"` |
| `gap`    | `number`  | Числовое значение, умножающееся на 0.25rem, чтобы задать отступ между элементами | `"2"` (результат: `0.5rem`) |
| `class`  | `string`  | Дополнительные CSS-классы                                                | `"my-flex"`             |

⚠️ **Важно**:
- Второе выравнивание (неосновное направление) всегда фиксировано в `center`.
- Если не указан атрибут `col` или `row`, по умолчанию используется `row`.

## Пример использования

```jsx
<Flex row>
  <Text>Первый элемент</Text>
  <Text>Второй элемент</Text>
</Flex>
