# Tooltip (Подсказки)

Любой компонент в DSL может быть обёрнут в тултип (подсказку) с помощью специальных атрибутов или слота.

---

## Поддерживаемые атрибуты

| Атрибут           | Описание                                                       | Пример                      |
|-------------------|----------------------------------------------------------------|-----------------------------|
| `tooltip:text`    | Текст тултипа, который будет отображаться при наведении        | `tooltip:text="Подсказка"`  |
| `tooltip:id`      | Явный ID тултипа (по умолчанию генерируется автоматически)      | `tooltip:id="my-tooltip"`   |

---

## Слот `#tooltip`

Вы можете определить **слот** с именем `tooltip`, чтобы передать кастомный HTML-контент тултипа.

```jsx
<Button>
  Нажми меня
  <template #tooltip>
    <div>Подсказка с <b>форматированием</b>!</div>
  </template>
</Button>
